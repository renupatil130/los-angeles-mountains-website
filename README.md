# Los Angeles Mountains Website

A responsive and visually rich static website inspired by the scenic beauty of the Los Angeles mountain ranges.  
This project focuses on clean layout design, smooth user experience, and modern front-end practices using only HTML, CSS, and JavaScript.

---

# Project Overview

The **Los Angeles Mountains Website** is a front-end–focused project that showcases a cinematic mountain-themed layout.  
It demonstrates how a well-structured static website can deliver an engaging visual experience without relying on heavy frameworks.

The project was built to strengthen core web development skills such as layout design, styling, and basic interactivity.

---

# Technologies Used

- **HTML5** – Structure and semantic layout  
- **CSS3** – Styling, layout, and responsiveness  
- **JavaScript** – Basic interactivity and behavior  
- **Images & Media Assets** – Visual enhancement


# Key Features

- Fully responsive design  
- Clean and minimal UI  
- Smooth navigation between sections  
- Optimized image usage  
- Lightweight and fast-loading pages
